Telegram Setup

  1. Open Telegram and search for @BotFather
  2. Send /newbot and follow the prompts
  3. Copy the bot token and paste it below
  4. Send /start to your new bot to get your chat ID

Discord Setup

  1. Go to discord.com/developers/applications
  2. Create a new application, then go to Bot section
  3. Click Reset Token and copy it below
  4. Under Privileged Gateway Intents, enable Message Content Intent
  5. Go to OAuth2 > URL Generator, select bot scope with Send Messages + Read Message History permissions, then use the generated URL to invite the bot to your server
